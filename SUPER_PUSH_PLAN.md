# 山巅微光超级推进计划

记录时间：2026-05-31

## 总判断

现在的目标不是“加几个功能”，而是把 `山巅微光` 推成一个可信的精密平台跳跃 demo。要超越参考作品，不能靠复刻，也不能只靠更难。真正的突破口是：

- 手感可信：玩家相信输入、碰撞、宽限和重开。
- 关卡可读：每个房间能看懂安全线、进阶线和高手线。
- 练习领先：死亡、split、ghost、Focus、Drill、medal 和复盘组成完整训练闭环。
- 原创动作语言：Spark hop、光继点、风、棱镜、回声和脆冰形成自己的节奏。
- 工程稳态：每次推进都能被脚本、浏览器 smoke 和文档约束住。

## 反思机制

每一批推进都必须先问四件事，再继续：

1. 这个改动是在提高“可练”和“可读”，还是只是在堆 UI？
2. 这个改动会不会让危险物、角色状态、ghost 或练习线互相遮挡？
3. 这个改动会不会让玩家误会计时、clean、PB 或 Focus 数据？
4. 这个改动能不能被质量门或浏览器 smoke 检查到？

如果答案不稳，就先缩小范围，改成更可验证的一步。

## 当前超级推进包

### A. 练习入口

- Coach 推荐房间不只是跳转，而是进入 Drill。
- Drill 必须显示目标，而不是让玩家猜为什么被推荐。
- Drill 完成后反馈 clean/clear，并保留真实错误记录。
- 主 Drill 入口必须解析为 Clean/Pace/Style/Expert 真实合同，避免最显眼的训练按钮变成宽松通关。已完成基础版。
- 设置面板提供 Clean/Pace/Style/Expert 四种 Drill 变体，分别对应无失误、target、难度类型和高手线。已完成基础版。
- Style Drill 必须让每房难度风味不同：精度、恢复、节拍、连锁、路线、地形风险、读图和终盘耐压都要能被练到。已完成基础版。
- Style 目标必须进到游玩时反馈：入场卡、Drill HUD 和登顶复盘都要能读到类型目标。已完成基础版。
- Style/Expert 缺失动作必须在房间里有信标提示，让玩家知道该去找哪个机制。已完成基础版。
- Expert Drill 必须证明本房高手动作，不允许只用快速 safe 线混过。已完成基础版。
- Drill 未达标时应直接重开同一房间，减少菜单往返和错误练习延续。已完成基础版。
- 复盘区提供 Clean/Pace/Style/Expert 四张训练卡，直接从数据生成下一步，并显示合约尝试/完成进度。已完成基础版。
- 复盘区提供全房间掌握表，按训练优先级排序并一键进入对应 Drill。已完成基础版。

### B. 房间掌握度

- 房间目录显示 medal、PB/target、clean、技能标签和 Focus 标记。
- 入场卡显示 medal/clean，让玩家一进房就知道本房练习状态。
- 后续继续补 room purpose、三层路线和目标提示。

### C. 复盘系统

- 登顶复盘显示 clean 房间、Flow、死亡结构、最大 split 损失、最弱房和下一 Drill。
- 登顶复盘拆成卡片，最大 split loss 和薄弱原因都能落到具体路线建议。已完成基础版。
- 登顶复盘可直接进入下一 Drill 或最慢房 Drill，减少菜单往返。已完成基础版。
- 后续继续补每房 split loss 表和低摩擦重练入口。

### D. 视觉可读性

- 练习线和 ghost 必须可开关、可调透明度。
- 角色新增跳跃、冲刺、Spark、墙跳、光继、棱镜、弹簧、召回、出生、落地和死亡的状态脉冲。已完成基础版。
- 脆冰碎裂进度、风场边界、光继/棱镜冷却环和回声召回牵引线已完成基础版。
- 后续继续重做更完整的角色动画帧和章节级环境变化。

### E. 质量门

- 所有新 UI id 必须进 contract check。
- 所有新房间数据必须长度对齐 maps。
- 每轮结束跑 `npm run check`。
- 前端变更必须做本地浏览器 smoke。
- 中文练习文案要用中英文分流的契约检查，避免为了通过英文长度阈值而写废话。

### F. 设计可读性

- 计时只记录真实游玩，不记录出生后犹豫和设置阅读时间。已完成基础版。
- 设置面板打开时暂停模拟，降低误操作和数据污染。已完成基础版。
- 设置面板按画面、练习、复盘分组，练习文案优先中文化。已完成基础版。
- 窄屏 HUD 收紧信息密度，触控控件只在触控设备显示。已完成基础版。

## 自主推进节奏

后续尽量不问用户，除非遇到以下情况：

- 需要删除或重写用户未要求删除的内容。
- 需要访问外部账号、发布、推送或执行不可逆操作。
- 出现两种都合理但产品方向相冲突的选择。

默认推进顺序：

1. 先增强训练闭环。
2. 再补房间目的和复盘解释。
3. 再做手感与地图微调。
4. 再做角色/特效视觉 pass。
5. 最后扩房间数量。

## 2026-06-01 严格 UX 审查补充

这轮审查结论：训练系统已经足够厚，但小窗口的第一眼读图还不够稳。后续所有新增信息必须先过“跑动中是否能读懂”的门槛，再考虑展示更多数据。

- 窄屏 HUD 必须保持单行和低遮挡，优先保留房间号、pace 差值、死亡数和三条状态条；重复计时和收集数可以在小窗口隐藏。
- 入场目标必须让开 HUD，并在小窗口加大字号和提高底层对比度；房间目的、Style 目标和技能标签要能被一眼读到。
- 视觉优化必须服务读图：目标文本、危险线、pace 条、Flow 氛围和练习信标不能互相抢焦点。
- 下一批体验方向：死亡后的即时复盘更短、更靠近死亡点；Drill 失败提示要更像动作建议；设置面板继续减少“表格感”，把下一步训练动作放在最前。
- 已推进：设置面板顶部增加“下一步”训练主卡；死亡/手动重开后显示单一死亡教练提示，并压掉重复 Focus 弹层，避免玩家同时读两份相同建议。

## 2026-06-02 全面体验审查补充

这轮重点不是加更多系统，而是把已经很厚的练习系统变得更可靠、更少误操作、更适合小屏持续练习。

- 设置按钮和 HUD 统计从 `aria-hidden` 中解放出来，并同步 `aria-expanded` / `aria-controls`；只有装饰性状态条对辅助技术隐藏。
- 设置打开时给画面增加暂停态压暗，避免玩家误以为角色还在真实运行。
- Focus/Drill 统计清空改为带过期时间的二次确认，Reset 第一次只进入 armed 状态，第二次才执行。
- 出生/重开后的计时保护增加画面提示，玩家能看见“松开待命 / 首次输入开始计时”的状态。
- 练习队列卡增加“开练”动作 affordance；窄屏下掌握表隐藏次要统计，优先保留房间、建议动作和一键入口。
- 质量门补充设置可访问性、计时提示、Reset 确认、暂停视觉态和训练卡动作提示，防止后续回退。

## 2026-06-02 训练计划超大步

目标是把“练什么”从复盘表格升级成连续训练流，让玩家少读菜单，多进入下一次有效尝试。

- 设置面板新增三步训练计划：现在修最短板，接着换一种能力，然后补路线链；每一步都是可点击 Drill。
- 死亡教练从原因提示升级为动作处方：下一次该怎么按、该走哪条线、推荐进哪个 Drill 会同时出现。
- Drill 失败文案改成合约诊断，直接说明慢多少、缺哪个动作或是否破坏无失误，并给下一次路线动作。
- Drill HUD 增加 Clean/Pace/Style/Expert 合同条，玩家在跑动中能看到当前房间掌握阶梯。
- 质量门补上训练计划、死亡处方和合同条，保证训练闭环不会被后续视觉或地图改动冲掉。

## 2026-06-02 手感可信超大步

目标是让玩家相信输入窗口，而不是把成功归因于运气、把失败归因于系统。

- 跳跃 apex 加入轻量重力塑形，跳顶更容易微调，不改变核心速度上限。
- 命中 coyote、jump buffer、wall grace、Spark 压线或冲刺方向记忆时，会在角色旁给出短反馈。
- 输入提示新增 apex 稳定线，调试面板显示 feel cue、apex 和 aim memory 状态，便于后续继续调手感。
- 质量门补充 feel cue、apex gravity 和调试可见性，防止手感反馈被后续改动冲掉。

## 2026-06-02 路线清晰度超大步

目标是让玩家少猜“下一步该怎么走”，并在完成后立刻知道自己是不是更强了。

- 房间入场卡显示当前 SAFE/FAST/EXPERT 推荐线、路线动作和 mastery 分数。
- 奔跑中加入低打扰路线卡，短时间显示本房推荐路线、pace 状态和掌握阶段。
- 角色附近加入路线指南针，指向下一个关键实体或出口，Drill 中常驻，普通跑动中短暂出现。
- 清房和 Drill 成功后弹出 mastery 反馈，显示分数增量、clean/PB/合同完成和当前评级。
- Debug 和质量门补上 route cue、route compass 与 mastery popup，防止路线可读性被后续迭代冲掉。

## 2026-06-02 掌握路线图超大步

目标是把复盘从“看数据”推进到“马上知道下一步点什么”。

- 终点复盘新增掌握路线图，按房间缺口排序展示下一批 Clean/Pace/Style/Expert 合同。
- 路线图每一行都是 Drill 入口，点击后直接进入对应房间和合同，不需要再回设置面板查表。
- 设置掌握表加入 C/P/S/X 合同 pill，区分完成、尝试、当前推荐和未开练。
- Drill 完成反馈和 mastery toast 都会追加下一步合同，例如“下一 Pace Drill”或“维护 PB / Flow”。
- 质量门补上路线图 helper、合同 pill 和 CSS，保证训练闭环不会退回纯文字复盘。
