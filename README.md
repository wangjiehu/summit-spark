# 山巅微光

原创平台跳跃原型，包含跳跃、冲刺、攀墙、体力、尖刺、收集物、快速重开、调试面板、计时练习反馈，以及非实体光继点。

## 运行

在线试玩：

```text
https://wangjiehu.github.io/summit-spark/
```

```bash
npm start
```

打开：

```text
http://127.0.0.1:4173/
```

也可以直接用浏览器打开 `summit-spark.html`。

## 操作

- 移动：A/D 或 ←/→
- 方向输入：W/S 或 ↑/↓（用于冲刺方向和攀墙）
- 跳跃：Space / C / J
- 冲刺：X / K / Shift（E 保留兼容）
- 抓墙：Z / L / Ctrl（V 保留兼容）
- 快速重开：R
- 调试面板：F3
- 设置面板：右上角按钮 / O

## 开发方向

核心目标不是复刻某个现有游戏。光痕目前作为纯视觉反馈保留；光继点承担受限续航职责，只奖励冲刺、高速或续跃状态，避免无限悬空。

当前手感重点：冲刺结束后有极短的续跃窗口，能把一次冲刺转成更高的上升和横向延伸；墙跳区分普通墙跳、反向墙跳和抓墙跳，让路线选择更细。

完整路线见 `ROADMAP.md`。
