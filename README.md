# 山巅微光

原创平台跳跃原型，包含跳跃、冲刺、攀墙、体力、尖刺、收集物、快速重开、调试面板、计时练习反馈、房间分段 PB、Flow 峰值、最佳路线影子、死亡轨迹回放、手柄输入，以及非实体光继点、风升流、超载棱镜、回声锚点。

## 运行

在线试玩：

```text
https://wangjiehu.github.io/summit-spark/
```

```bash
npm start
```

打开：

```text
http://127.0.0.1:4173/
```

也可以直接用浏览器打开 `summit-spark.html`。

## 操作

- 移动：A/D 或 ←/→
- 方向输入：W/S 或 ↑/↓（用于冲刺方向和攀墙）
- 跳跃：Space / C / J
- 冲刺：X / K / Shift（E 保留兼容）
- 抓墙：Z / L / Ctrl（V 保留兼容）
- 快速下落：下方向 / S
- 回声召回：Q / Backspace（触碰回声锚点后生效）
- 键位预设：设置里可切换舒适 / 经典；经典会把 Z 作为跳跃、C 作为抓墙
- 快速重开：R
- 当前房间重开：T
- 调试面板：F3
- 设置面板：右上角按钮 / O
- 练习房间：设置里的房间下拉框可直接跳到指定房间
- 手柄：左摇杆 / 十字键移动，A 跳跃，B/X/RT 冲刺，肩键 / LT 抓墙，Y / Back 召回

## 开发方向

核心目标不是复刻某个现有游戏。光痕目前作为纯视觉反馈保留；光继点承担受限续航职责，只奖励冲刺、高速或续跃状态，避免无限悬空。

当前手感重点：冲刺结束后有极短的续跃窗口，能把一次冲刺转成更高的上升和横向延伸；墙跳区分普通墙跳、反向墙跳和抓墙跳，并带短墙面宽限；快速下落能压低滞空；跳、冲、抓、下落都有贴身输入回声；风升流改变空中节奏，超载棱镜把高速路线推向更高上限，回声锚点减少练习摩擦；脆冰平台会在后段短暂承重后碎裂，迫使玩家按路线节奏推进；连续触发光继点会出现连锁反馈；每个房间会记录最佳 split、评级、Flow 峰值和最佳路线影子。

完整路线见 `ROADMAP.md`。

## 最新推进

- 死亡诊断：尖刺、坠落、脆冰节奏、手动重开都会被记录并显示在死亡标记和调试面板里。
- 难度曲线：7-10 房间加入脆冰平台和更密的机制组合，后段压力明显提高。
- split 结果：过房间时会弹出 PB/SPLIT 和差值，每段练习立刻有反馈。
- split delta：HUD 直接显示当前房间相对 PB/target 的差值。
- 冲刺意图：角色旁的细线预览当前冲刺方向，斜向和记忆方向更清楚。
- 练习房间：设置面板新增房间选择，可直接跳段练习。
- 地图体检：`npm run check` 会验证房间尺寸、tile、target、名字、起点、终点和入口/出口缺口。
- 练习线：可在设置里开关，显示当前路线轨迹和最佳幽灵。
- 动态最佳幽灵：按当前 split 时间推进，用来追高手路线。
- 房间入场卡：显示房间名、target 和 best，让练习节奏更清楚。
- 登顶复盘：终点增加房间掌握度和 Flow Best 汇总。
