# 山巅微光

原创平台跳跃原型，包含跳跃、冲刺、攀墙、体力、尖刺、收集物、快速重开、调试面板、计时练习反馈、房间分段 PB、Flow 峰值、最佳路线影子、死亡轨迹回放、房间 Focus 教练、Drill 练习、手柄/触控输入，以及非实体光继点、风升流、超载棱镜、回声锚点。

## 运行

在线试玩：

```text
https://wangjiehu.github.io/summit-spark/
```

```bash
npm start
```

打开：

```text
http://127.0.0.1:4173/
```

也可以直接用浏览器打开 `summit-spark.html`。

## 操作

- 移动：A/D 或 ←/→（触控现在也有上/下/左/右）
- 方向输入：W/S 或 ↑/↓（用于冲刺方向和攀墙）
- 跳跃：Space / C / J
- 冲刺：X / K / Shift（E 保留兼容）
- 抓墙：Z / L / Ctrl（V 保留兼容）
- 快速下落：下方向 / S
- 回声召回：Q / Backspace（触碰回声锚点后生效）
- 键位预设：设置里可切换舒适 / 经典；经典会把 Z 作为跳跃、C 作为抓墙
- 快速重开：R
- 当前房间重开：T
- 调试面板：F3
- 设置面板：右上角按钮 / O
- 练习辅助：设置里可开关练习线，并调节最佳路线/ghost 透明度
- 练习目录：房间下拉框显示 target/PB、相对 target 损失、评级、clean、Drill、技能标签和房间失误标记（`!x` / `watch`），更容易找薄弱房间。
- 练习教练：设置面板会推荐当前最该练的房间，Drill 按钮可直接进入目标练习，Reset 需要二次确认后才会清空 Focus/Drill 统计。
- Drill 状态：练习中会显示当前房间目标和本轮错误数；主 Drill 会自动解析为 Clean/Pace/Style/Expert 合同；四种变体可分别练无失误、target、房间难度类型和高手线；Expert 还会检查本房标志性动作是否真的做到了；失败会直接重开同一 Drill；Drill start/clear/clean 会写入每房统计。
- 房间详情：设置面板会展示当前房间目的、safe/fast/expert 路线、pace、clean 和 Drill 信息。
- 练习报告：设置面板会汇总 clean 进度、Drill 完成情况、Clean/Pace/Style/Expert 合约进度、训练航线、薄弱房、最大 split 损失和下一 Drill。
- 练习课表：设置面板会自动生成 Clean、Pace、Style、Expert 四张训练卡，显示每种合约的尝试/完成和进度，可一键开始对应房间 Drill。
- 房间掌握表：复盘区按优先级列出全部房间，显示 mastery 分数、pace/clean/contract 状态和建议 Drill，点击任意行可直接重练。
- 登顶复盘卡：结束画面会把下一 Drill、最大 split 损失、薄弱原因和训练航线拆成卡片，并给出具体路线建议；按钮可直接进入下一 Drill 或最慢房 Drill。
- 手柄：左摇杆 / 十字键移动，A 跳跃，B/X/RT 冲刺，肩键 / LT 抓墙，Y / Back 召回

## 开发方向

核心目标不是复刻某个现有游戏。光痕目前作为纯视觉反馈保留；光继点承担受限续航职责，只奖励冲刺、高速或续跃状态，避免无限悬空。

当前手感重点：冲刺结束后有极短的续跃窗口，能把一次冲刺转成更高的上升和横向延伸；墙跳区分普通墙跳、反向墙跳和抓墙跳，并带短墙面宽限；快速下落能压低滞空；跳、冲、抓、下落都有贴身输入回声；风升流改变空中节奏，超载棱镜把高速路线推向更高上限，回声锚点减少练习摩擦；脆冰平台会在后段短暂承重后碎裂，迫使玩家按路线节奏推进；连续触发光继点会出现连锁反馈；每个房间会记录最佳 split、评级、Flow 峰值、Focus 画像、Drill 统计和带时间轴的最佳路线影子。

完整路线见 `ROADMAP.md`；长期优化总纲见 `LONG_TERM_OPTIMIZATION_OUTLINE.md`；超级推进计划见 `SUPER_PUSH_PLAN.md`。

## 最新推进

- 死亡诊断：尖刺、坠落、脆冰节奏、手动重开都会被记录并显示在死亡标记和调试面板里。
- 难度曲线：第 5 房改为上下折返练习，第 7 房安全引入风+脆冰，第 8 房把棱镜改成路线选择器，第 9-10 房按锚点/棱镜/中继分幕收束。
- split 结果：过房间时会弹出 PB/SPLIT 和差值，每段练习立刻有反馈。
- split delta：HUD 直接显示当前房间相对 PB/target 的差值。
- 冲刺意图：角色旁的细线预览当前冲刺方向，斜向和记忆方向更清楚。
- 练习房间：设置里的房间选择可直接跳段练习。
- 地图体检：`npm run check` 会验证房间尺寸、tile、target、名字、起点、终点、入口/出口缺口、技能标签、Focus 契约、HTML 双文件一致和部署前质量门。
- 练习线：可在设置里开关，显示当前路线轨迹、静态最佳线和最佳幽灵。
- 动态最佳幽灵：保存路线时间戳并按真实房间时间插值，用来追高手路线。
- 房间入场卡：显示房间名、target、best、pace、clean、Drill 和房间目的。
- 登顶复盘：终点增加房间掌握度、Flow Best、Drill 统计、最大 split 损失和下一练习建议。
- 薄弱房间画像：尖刺、坠落、脆冰节奏、手动重开都会写入每个房间的练习统计，并在入场卡、调试面板、练习目录和通关复盘里回显。
- 练习增强：Drill 入口会把推荐房间变成明确目标；每房有 safe/fast/expert 路线目标；房间详情、目录和入场卡显示 medal/pace/clean/Drill 状态与房间目的；最佳路线/ghost 透明度可调。
- 设计打磨：计时改为首次操作后启动，设置面板打开时暂停模拟；练习文案中文化，设置面板按画面/练习/复盘分组，窄屏 HUD 更克制，触控按钮只在触控设备显示。
- 视觉复盘大步：角色新增跳跃、冲刺、Spark、墙跳、光继、棱镜、弹簧、召回、落地的状态脉冲；练习报告和登顶复盘会直接说明“为什么练这一房”和“先练哪条路线”。
- 后段机制可读性：脆冰增加碎裂进度和危险描边，风场增加范围边界与上升箭头，光继/棱镜增加冷却环，回声锚点会在可召回时连到角色。
- Drill 变体：设置面板提供 Clean、Pace、Style、Expert 四个按钮；Style 绑定每房不同难度类型，例如落点精度、恢复控制、弹簧节拍、连锁节奏、地形风险、回声读图和终盘耐压；Pace/Expert 会按房间 target 和无失误条件判断是否真正达标。
- 练习课表：复盘区新增 Clean/Pace/Style/Expert 训练卡，卡片会显示合约尝试/完成进度，并把每种合约的 start/win 写入每房 Focus 档案。
- 掌握表：复盘区新增全房间 mastery 表，按训练优先级排序并支持一键进入对应 Drill。
- Expert 合约：高手线不再只是 S + 无失误，还要求对应房间的关键动作，例如 Spark、光继连锁、弹簧、风、棱镜、回声或脆冰节奏。
- Drill 重练：合约未达标时会回到本房起点继续练，不再把失败练习带到下一房。
- 主 Drill 合同化：设置面板和终点复盘里的“下一 Drill”会显示并启动当前最需要的 Clean/Pace/Style/Expert 合同，不再走只要到出口就通过的宽松 auto 练习。
- Style 合同：每个房间拥有一个专属难度类型挑战，成功会检查对应动作、部分房间的无失误要求和较宽松时间窗，让训练从“更快”扩展到“不同玩法能力”。
- 类型反馈：房间入场会显示 Style 目标；Style/Expert/Pace Drill HUD 会显示动作进度或剩余时间；登顶复盘会单独推荐一个类型挑战并可一键进入。
- 目标信标：Style/Expert Drill 会在关卡里高亮还没完成的关键动作目标，例如光继、弹簧、风、棱镜、回声、召回、脆冰或 Spark。
- 计时保护：出生/重开后必须先出现一次空输入，再由真实操作启动计时，降低手柄漂移或残留按键污染 split 的风险。
- UX 审查补强：设置按钮和 HUD 统计不再被可访问性隐藏；设置打开时会视觉暂停画面；出生后的计时待命提示、训练卡“开练” affordance、窄屏掌握表和 live status 都已补强。
- 训练计划：设置面板会自动生成“现在 / 接着 / 然后”三步 Drill 路线；死亡提示会给出下一次动作处方和推荐 Drill；Drill HUD 会显示 Clean/Pace/Style/Expert 合同条，帮助玩家看到掌握阶梯。
- 手感可信：跳跃 apex 加入轻量重力塑形；buffer、coyote、墙宽限、Spark 和冲刺方向记忆都会在命中时给出贴近角色的短反馈，调试面板也会显示当前 feel 窗口。
- 路线清晰度：房间入场卡、奔跑中的路线卡和贴身指南针会显示当前 SAFE/FAST/EXPERT 推荐线、下一目标实体和掌握度；清房或 Drill 完成时会弹出 mastery 进步反馈。
