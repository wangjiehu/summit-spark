# 山巅微光

原创平台跳跃原型，包含跳跃、冲刺、攀墙、体力、尖刺、收集物、快速重开、调试面板，以及更清晰的冲刺光痕反馈。

## 运行

在线试玩：

```text
https://wangjiehu.github.io/summit-spark/
```

```bash
npm start
```

打开：

```text
http://127.0.0.1:4173/
```

也可以直接用浏览器打开 `summit-spark.html`。

## 操作

- 移动：A/D 或 ←/→
- 方向输入：W/S 或 ↑/↓（用于冲刺方向和攀墙）
- 跳跃：空格
- 冲刺：X / E / Shift
- 抓墙：C / Z / Ctrl
- 快速重开：R
- 调试面板：F3
- 设置面板：右上角按钮 / O

## 开发方向

核心目标不是复刻某个现有游戏。光痕目前作为纯视觉反馈保留，后续会把空中造路、连锁冲刺、限时落点和隐藏路线重做成更受控、更有风险回报的原创机制。
