# 山巅微光

原创平台跳跃原型，包含跳跃、冲刺、攀墙、体力、尖刺、收集物、快速重开、调试面板，以及冲刺后生成可踩光痕的平台机制。

## 运行

在线试玩：

```text
https://wangjiehu.github.io/summit-spark/
```

```bash
npm start
```

打开：

```text
http://127.0.0.1:4173/
```

也可以直接用浏览器打开 `summit-spark.html`。

## 操作

- 移动：方向键 / WASD
- 跳跃：空格 / W / 上方向键
- 冲刺：X / E / Shift
- 抓墙：C / Z / Ctrl
- 快速重开：R
- 调试面板：F3
- 设置面板：右上角按钮 / O

## 开发方向

核心目标不是复刻某个现有游戏，而是围绕“光痕平台”形成自己的关卡语法：空中造路、连锁冲刺、限时落点和隐藏路线。
